## 2024-05-18 - Keyboard Accessibility for Hover-Only Actions
**Learning:** In the `TaskDashboard` component, task action buttons (Archive, Delete) were initially set to be visible only on hover using `group-hover:opacity-100`. This made them completely inaccessible to keyboard users, as tabbing into the hidden container did not make the buttons visible. Furthermore, these icon-only buttons lacked clear visual focus indicators.
**Action:** Always ensure that UI elements or buttons that appear exclusively on hover (e.g., via `group-hover:opacity-100`) also include focus-visible styles. Use `focus-within:opacity-100` on the container to ensure the entire group becomes visible when keyboard focus enters it. Additionally, explicitly define focus states for individual buttons using classes like `focus-visible:ring-2` to provide clear feedback to keyboard users.
