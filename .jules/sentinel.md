## 2024-03-03 - Deferred DOM-based XSS via innerHTML rendering array
**Vulnerability:** Deferred DOM-based XSS in scroll animations (useScrollAnimations.tsx)
**Learning:** React refs or native DOM queries capturing `element.innerText` to extract text from an element, splitting it, and wrapping each chunk with raw HTML strings formatted into `element.innerHTML` poses a DOM-based XSS vulnerability. If the text originally injected inside `.reveal-text` element is user-generated without sanitization, malicious code executing `<script>` or `<img onerror=...>` escapes the bounds of text due to raw interpolation in `innerHTML`. Because this triggers on intersection observations, it poses an unexpected deferred payload execution upon scrolling.
**Prevention:** Avoid `element.innerHTML` for wrapping text in markup. Manually create DOM objects via `document.createElement`, assign properties and standard APIs safely, and populate text specifically through `.textContent`. Empty existing parent containers carefully and append safely using `.appendChild`.
